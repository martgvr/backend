# Repository usage

## How to use this repository
To start using this repository execute one of the following commands

```
$ npm start
```
```
$ npm test
```

## Environment Variables
An environment file (.env) should be placed at the repository's root folder

```
MONGO_URL = ''
SECRET_KEY = ''
CLUSTER = [true | false]
PORT = [number]
DAO = [file | mongo]
ADMIN_EMAIL = ''
NODEMAILER_USER = ''
NODEMAILER_PASS = ''
```
